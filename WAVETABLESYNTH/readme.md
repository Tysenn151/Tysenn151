Audio Programming and DSP

The Computer Music Tutorial by Curtis Road
- directed toward musicians


Discrete-Time Signal Processing by Alan Oppenheim
- reference

Digiatl Audio FX (DAFX) edited by Udo Zolzer
- source code for audio effects
- reference
- has a conference by the same name


The C++ Programming Language by Bjarne Stroustrup
- realtime audio projects are often in C++
- this is the author of the language
- 1300 pages
- first 5 chapters give you exhaustive tour of language and ready to start
- its from C++ 11, not the latest


Clean Code by Robert C. Martin
- best practices for software engineer
- Wolfsound has a condensed interpretation of it


Design Patterns: Eelements of Reusable Object Oriented Software by GoF
- best practices

A mind for numbers by Barbara Oakley
- manual to the brain, and how to learn
- teaches you how to learn
- especially if you're struggling with learning topics related to science


Introduction to Digital Filters with Audio Applications by Julius O. Smith III
- available for free on his website, with other books (julius's)
- written for students

Operating Systems: Internals and Design Principles by William Stallings
- how Windows/Linux/MacOS, etc work
- step by step how to build a whole operating system
- it's important to understand threads, processes and synchronization... this feeds back to Audio
